# Drinks Galore

An application that uses TheCocktailDB to diplay an multitude of alcoholic and non-alcoholic drinks. Random drinks are provided to the user through the drink of the day option.

## Languages and Frameworks

### Front-End
CSS
React
Bootstrap
React-Routers
TheCocktailDB API
